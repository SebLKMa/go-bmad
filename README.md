# go-bmad
A hello world for Spec-Driven Development using [BMAD](https://github.com/bmad-code-org/bmad-method) with claude.

## Quick steps

After installation and in Claude, run `bmad-quick-dev`.  

```sh
Build a command line hello-world program to say hello arg where arg is a command line arg
```
